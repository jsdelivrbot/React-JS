# TMDb Movie Search
This is a Movie Search app that utilises Twitter's [typeahead.js](https://twitter.github.io/typeahead.js/) and [Bloodhound](https://github.com/twitter/typeahead.js/blob/master/doc/bloodhound.md) suggestion engine, loading data via [The Movie Database (TMDb) API](https://www.themoviedb.org/documentation/api).

## Tools
Key tools used in this ReactJS project are:

* [ReactJS](http://facebook.github.io/react/index.html)
* [Typeahead.js](https://twitter.github.io/typeahead.js/)
* [Bloodhound](https://github.com/twitter/typeahead.js/blob/master/doc/bloodhound.md)
* [Bootstrap](http://getbootstrap.com/)
* [SASS](http://sass-lang.com/)
* [Browserify](http://browserify.org/)
* [Babel](https://babeljs.io/)
* [Gulp](http://gulpjs.com/)
